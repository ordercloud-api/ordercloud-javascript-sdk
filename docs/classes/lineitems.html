      ID of the order.

      • Optional filters?: Filters

SetShippingAddress

  • SetShippingAddress<TLineItem>(direction: OrderDirection, orderID: string, lineItemID: string, address: Address, requestOptions?: RequestOptions): Promise<RequiredDeep<TLineItem>>

  • Set a shipping address. Set the Shipping Address of the Line Item. Appropriate only when the Address is not to be saved/reused. To use a Saved Address (i.e. from the Addresses resource), PATCH the Line Item's ShippingAddressID property instead. Check out the api docs for more info

      Direction of the order, from the current user's perspective. Possible values: incoming, outgoing.

    • orderID: string

      ID of the order.

    • lineItemID: string

      ID of the line item.

    • address: Address

      Required fields: Street1, City, State, Zip, Country

    • Default value requestOptions: RequestOptions = {}

    Returns Promise<RequiredDeep<TLineItem>>
